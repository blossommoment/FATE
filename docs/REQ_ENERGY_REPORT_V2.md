> 范围：`lib/fate.ts` 算法层 + 事实清单结构 + 报告章节组织。**不动 UI 视觉**（保持编辑风原设计）。

## 一、问题诊断（对应现状代码）

### 1.1 五行能量算法：量的是「出现量」，不是「旺衰」

现状（`calculateBazi`，fate.ts:120-135）：

```
天干权重 [5, 5, 0, 5]（日主不计）
藏干权重 年[8,5,2] 月[20,10,5] 日[8,5,2] 时[10,6,4]
→ 累加后归一化为百分比
```

缺陷：

| 缺陷 | 后果（以卯月丁火为例） |
|---|---|
| 无季节系数 | 卯月只给木 +20，火拿不到「春木生火」的相位加成 → 丁火被误判为弱 |
| 无生扶传导 | 印星当令生身这一最重要的强弱来源完全不参与计算 |
| 合冲不改能量 | 子午冲后两支能量原样计入；六合绊住的支照常全额贡献 |
| 无距离衰减 | 年支与时支的遥冲、紧贴的冲同权处理（合盘部分有 strength，个人盘没有） |

### 1.2 报告同质化：同一批数据被七个章节各自复述

合盘的 dimensions / behaviors / dispositions / frictions / verdict 都从同一组
personality 分数和十神权重取材，各章节独立成文，无「事实归属」约束——
表达差、依恋类型、主动性这几组数字会在贰、肆、伍章反复出现且结论雷同。

### 1.3 逻辑性弱：有目录框架，无论证主线

七章是「平铺的抽屉」，不是「展开的论证」。读者看完记不住这份报告到底说了
一件什么事，因为报告自己也没有回答「这段关系/这张命盘的核心命题是什么」。

## 二、五行能量引擎 v2（新模块 `lib/energy.ts`）

### 2.1 设计原则

- **口径**：以子平法「得令、得地、得势」三要素定强弱；季节旺衰用**五行级的
  旺相休囚死**，不用阴干逆行十二长生（理由见 2.6 流派说明）。
- **可追溯**：每一步加减都产出 source 记录（沿用 tenGodSources 的思路），
  报告任何结论都能引用「因为卯月火相 ×1.2」这样的依据——这同时服务问题三（逻辑性）。
- **与 v1 并行**：v2 输出挂在新字段上，报告切换后 v1 字段保留一个版本再删。

### 2.2 计算管线（五步）

```
① 基础分 → ② 合冲刑害预处理（改地支可用性）→ ③ 月令系数 → ④ 通根与透干 → ⑤ 生克传导定强弱
```

**① 基础分（存在量）**

| 位置 | 分值 |
|---|---|
| 天干（含日主，日主计入比劫党） | 50 |
| 藏干本气 / 中气 / 余气 | 100 / 50 / 30 |

> 天干为苗、地支为根：天干本体分低，强弱几乎取决于通根（见④）。
> 校验口诀：三个虚浮天干（3×15=45）< 一个本气根（100）；
> 两个虚浮天干（30）≈ 一个余气（30）——「得三天干不如一地支」成立。

**位置系数：不加**（已拍板）。月令的分量由季节系数 + 三处特权（三合
转化率、合化判定、六冲旺衰判向）承担，再乘位置即双重计分。若校准期
发现年支根被普遍高估，位置系数作为备选参数重新评估。

> 单项贡献公式：基础分 × 季节系数 × 可用系数（②的合冲结果）；
> 天干另乘通根系数（④）。

**② 合冲刑害预处理**——先处理合会，再处理冲，最后刑害；每支维护一个
「可用系数」availability（初始 1.0），最终该支藏干贡献 = 基础分 × availability（成局时部分转化为局五行）：

| 结构 | 判定 | 能量处理 |
|---|---|---|
| 三会 / 三合（全） | 三支齐 | 三支藏干贡献的 50% 转记为局五行；月令为局中支时转化率 70% |
| 半合 / 半会 | 缺一支 | 转化率 25%，且必须含旺支（子午卯酉）才成立 |
| 六合·合化 | 化神=月令五行，或化神在天干透出 | 两支的 60% 转为化神五行 |
| 六合·合绊 | 不满足化条件 | 两支 availability ×0.75（互相牵制，能量在但不出力） |
| 六冲 | 两支对冲 | 旺者（得月令方）×0.85，衰者 ×0.6；无明显旺衰则各 ×0.75；**众冲全额叠加**（已拍板不采「双冲不冲」），多冲可拔根 |
| 刑 / 害 | — | availability ×0.9，主要产出叙事标签；**冲刑同对不叠罚**（寅申、丑未首论冲，刑仅作标签） |

**活根阈值（根拔机制）**：根支可用性 < 0.45 即「根拔」——残余能量照算，
但不再阻挡从格判定、不再满足「身强须有根」，从格否决条件中被拔的支也
不再算「异物」。由此**冲致从格**成立：唯一禄根被双冲紧贴夹拔且无印相
救者，顺势入从；有印透干通根者守住身弱（杀印相生不弃命）。
| 天干五合 | 甲己/乙庚/丙辛/丁壬/戊癸相邻 | 化神得月令或有强根则化（60% 转化），否则合绊：两干 availability ×0.8 |

**距离衰减**（作用强度乘数）：紧贴 1.0，隔一柱 0.5，隔两柱（年—时）0.25。
同一支被多个结构作用时：合冲同现先论合（合可解冲，冲力减半而非全免）。

**③ 月令旺相休囚死系数**——按月支当令五行，对全盘五个五行分别乘：

| 相位 | 定义 | 系数 |
|---|---|---|
| 旺 | 当令者 | ×1.5 |
| 相 | 令所生 | ×1.2 |
| 休 | 生令者 | ×0.9 |
| 囚 | 克令者 | ×0.7 |
| 死 | 令所克 | ×0.6 |

土的特殊处理：辰戌丑未月土旺；寅卯月土死、巳午月土相沿用通则。

> 系数不宜过狠：藏干体系本身已编码大量季节信息（巳里藏庚=金在巳有气，
> 卯里纯乙=金在卯无根），季节系数与藏干若同时重罚会双重扣分。
> 死 0.6 / 囚 0.7 为初始值，校准时优先向上微调。

**可选精度开关：人元司令分野**——月内并非单一五行当令（如巳月立夏后
戊土→庚金→丙火分段司令）。开启后按出生日距节气的天数取实际司令五行
定「旺」，生于庚金司令段则金以有令论。lunar-javascript 可取节气日期，
分日数各派有异，默认关闭，留待校准期决定是否启用。

**④ 通根与透干**

根的层级（天干按最强的一个根取乘数）：

| 根 | 判定 | 天干乘数 |
|---|---|---|
| 禄刃根 | 地支本气 = 该干五行（临官/帝旺之地） | ×1.3 |
| 长生根 | 阳干的长生地（甲亥、丙戊寅、庚巳、壬申） | ×1.15 |
| 中气/余气根 | 地支中气或余气 = 该干五行 | ×1.1 |
| 墓库根 | 该干五行之墓（辰戌丑未对应） | ×1.05，另出叙事标签 |
| 虚浮（无根） | 全盘无根 | ×0.3 |

- 地支本气透干：该支本气贡献 ×1.2（干支一体，力量成形）。
- 十二长生的使用边界：阳干长生、禄刃、墓库入根级参与计算；
  阴干逆行表**不参与强弱计算**，仅作取象标签（理由见 2.4）。

**⑤ 生克传导与日主强弱**

- 同党 = 比劫 + 印星能量；异党 = 财 + 官杀 + 食伤能量。
- 印星传导：印党能量的 60% 视为对日主的有效生扶（印需有力才能生身，
  印自身被冲绊时传导率随 availability 下降）。
- **母旺子衰上限**：印党能量超过比劫党 2 倍的部分，传导率降为 30%
  （木多火塞——卯月丁火若满盘皆木，不封顶会把印重身轻误判成身强）。
- **从格置信度**：从强/从弱阈值 ±4% 区间内 confidence 强制降为 low，
  报告措辞软化为「趋近从势」——从格判反则喜忌全盘颠倒，宁可示弱不硬断。
- 强弱分 = 同党有效能量 / 全盘能量，结合得令与否分档：

| 档位 | 条件（示意，最终以黄金用例校准） |
|---|---|
| 从强 | 同党 ≥ 90% 且得令且异党无活本气根（会局成员随局论）；80~90 为「趋近从强」降置信带 |
| 身强 | 同党 ≥ 55% 且有活根（「45%+得令」条款已删——得令已计入季节系数） |
| 中和 | 30% ~ 55%（2026-07-03 扩宽：此量表异党三家同党两家，经验中位数 35%，中和取中点两翼） |
| 身弱 | 同党 < 30% |
| 从弱 | 同党 ≤ 20% 且无强根 |

> **无根不言强**：日主虚浮时即便党势占优（如印重身轻）也最高判中和。
> **从格否决条件（2026-07-03 收紧）**：从格容不得异物——从弱另须同党
> （比劫+印）在地支无本气根；从强另须异党（财官食伤）在地支无本气根。

**分布体检记录**（17,568 张 1950–2010 真实日期盘，61年×12月×6日×4时辰）：

| 档位 | 初版 | 收紧从格后 | 合理带 |
|---|---|---|---|
| 从弱 / 从强 | 6.7% / 4.7% | 3.9% / 1.8% | 从格合计 < 5% |
| 身弱 / 身强 | 49.3% / 30.7% | 52.2% / 33.6% | 弱多于强，符合现代主流观感 |
| 中和 | 8.6% | 8.6% | — |

待裁决：真从/假从之辨——交由公案盘与 AI 陪审团裁决，不再手工调系数；
临界盘一律 confidence low + 措辞软化兜底。具体切面：
① 印比虚浮透干是否破从格；
② 印透且有**中气根**是否破从格（现行否决只查本气根；实例：辛酉 丙申
   甲寅 壬申 判从弱 18.1%，严格派或判身弱极而不从）。

**历法有效性纪律（2026-07-03）**：判定级黄金用例必须通过五虎遁/五鼠遁
校验（真实历法可产生），已入测试守卫。教训：甲日申时必为壬申——历法
结构会强制某些干支同现（如甲寅日申月申时必带壬印），合成盘可能锁定
现实中不存在的结构。纯机制单测允许合成干支。冲致从格在真实历法中确有
实例（225,744 张扫描：根拔 0.12%，其中从弱 148 张，含三酉冲卯根的
1921-09-19 盘，已入黄金用例）。

- 输出 `dayMasterStrength: { score, level, confidence, reasons[] }`，
  confidence 低（临界区）时报告措辞自动软化（「偏向身弱」）。
- 顺带产出**用神/喜忌**（扶抑法为主）：身弱取印比为喜，身强取财官食伤为喜，
  从格顺势——它将成为深度分析报告的锚（见 3.2）。

### 2.3 卯月丁火验证用例（写进黄金测试）

```
丁火生卯月：
- 月令卯，当令五行为木 → 火处「相」位 ×1.2（春木生火）
- 卯中乙木为丁之偏印，本气 100 分 × 旺 1.5 → 强印
- 印星传导 60% 生身
结论：即便原局火只有日主一位，丁火也判「偏弱但有源」而非「弱」；
     若再得一个午/巳根或比劫透干，直接入中和~偏强。
```

**巳月庚金验证用例（同批必测）**

```
庚金生巳月：
- 朴素旺相休囚死：夏月金死 ×0.6 → 若只看这一层会误判「很弱」
- 修正一：巳中藏庚（中气）→ 天干庚通根月令，非虚浮（×1.1 起）
- 修正二：庚长生在巳 → 取「长生根」×1.15，弱而有生机
- 修正三（开关开启时）：若生于庚金司令段，金直接以有令论
结论：判「偏弱、进气、有根」，而非「很弱」。
```

同批用例建议再加：子午紧贴冲 vs 年时遥冲（衰减差异）、亥卯未全局
（三合转化）、六合绊 vs 合化（子丑合在丑月 vs 午月）、从弱格边界、
三虚浮天干 vs 一本气根（干支权重校验）。

### 2.4 流派说明：为什么不用阴干逆行十二长生算强弱

按阴干逆行表，丁长生在酉，卯位恰是「病」——照它算，卯月丁火反而更弱，
与「木旺生火」的常识矛盾（这正是你举这个例子戳中的点）。传统上十二长生
的阴干逆行争议极大（《渊海子平》用，《子平真诠》明确批评「阳长生有力，
阴长生不甚有力」），现代主流实践以**月令旺相休囚死 + 通根 + 生扶党势**
定强弱。v2 采用这个口径，但阳干的长生、禄刃、墓库以「根的层级」参与
计算（见④，巳月庚金即靠长生根救回），阴干逆行表只作取象标签。

### 2.5 明确不做（本期）

- 调候用神（冬火夏水的寒暖燥湿修正）——留作 v2.1 可选项。
- 湿土燥土之分（丑辰生金晦火、未戌脆金助火）——影响生克传导质量，
  留作 v2.1，第一版控制变量数量以便校准。
- 格局法定用神（正官格、伤官配印等）——现有「特殊格局」检测保留为标签，不参与能量计算。
- 神煞对能量的加减——神煞只做叙事。

## 三、报告重构：先立主线，再分章节

### 3.1 核心机制一：报告主线（spine）

事实清单新增顶层字段 `spine`，由规则引擎产出，是整份报告的「论点」：

```jsonc
// 合盘
"spine": {
  "thesis": "一段由「表达译码」驱动、被「主导权协商」考验的关系",
  "primaryResource": { "key": "expression", "why": "六维最高 + 双方食伤结构互补" },
  "primaryTension":  { "key": "power", "why": "六维最低 + 官杀权重差 + 日支冲" },
  "throughline": "每章须回答：这一章的内容如何佐证/展开/化解这条主线"
}

// 个人深度分析
"spine": {
  "thesis": "身弱印旺之局：输入远大于输出",
  "strength": { "level": "身弱", "score": 38, "confidence": "high" },
  "favorable": ["木", "火"],
  "coreTension": "印重身轻——吸收多、表达出口窄"
}
```

主线的选取规则是确定性的（六维最高/最低 + 最强跨盘结构 + 强弱档位的组合
映射到判词库），同输入同输出，符合 REPORT_STANDARD 的事实契约。

### 3.2 核心机制二：事实所有权（治重复）

事实清单里每条事实增加 `owner`（主场章节）字段。渲染约束：

- 事实只在 owner 章节完整展开（结论 + 依据 + 数字）；
- 其他章节引用时只允许短引（「如贰章所说的慢热结构」），**不得重复数字与依据**；
- 新增回归测试「重复度检查」：抽取各章节输出句子，任意两章的连续 12 字
  以上重合即失败（判词标题、人名豁免）。

### 3.3 合盘七章的职责契约（重排为论证结构）

| 章 | 职责（回答的问题） | 独占事实 | 禁区 |
|---|---|---|---|
| 壹·关系总览 | 提出主线：这段关系是怎样一个命题？ | spine、总分、判词、六维雷达（只给图和一句话） | 不展开任何单维分析 |
| 贰·两人底色 | 主线的成因之一：两个什么样的人相遇了？ | dispositions、依恋倾向、各自主轴十神 | 不谈互动，只谈个体 |
| 叁·八字化学反应 | 主线的命盘证据：两张盘怎么咬合？ | branchDynamics、五行互补、能量对比（用 v2 引擎） | 不重复个体性格 |
| 肆·相处样态 | 主线在日常的样子 | behaviors（谁主动/醋意/饭桌氛围） | 不预警风险 |
| 伍·摩擦与化解 | 主线中张力的场景化 | frictions/hotspots，且第一条必须是 primaryTension 的场景 | 不重复肆章的日常描述 |
| 陆·长线经营 | 针对主线的行动 | initiator、行动建议、约会灵感（每条注明化解哪个摩擦） | 不出现新结论 |
| 柒·总结 | 回扣主线 | 判词回顾 + 三句话收束（模板化，仅引用不新增） | 篇幅上限 150 字 |

阅读动线即论证动线：**提出命题 → 人的成因 → 盘的证据 → 日常表现 → 张力场景 → 行动 → 收束**。

### 3.4 个人深度分析：从「按数据源分章」改为「按问题分章」

现状章节按十神四组 + 专项观察排布（按引擎的数据结构分章）。改为按读者的问题：

| 章 | 问题 | 内容来源 |
|---|---|---|
| 壹·命局主线 | 我这张盘的核心命题是什么？ | spine：强弱 + 用神 + 核心张力，一页讲透 |
| 贰·能量结构 | 为什么是这个命题？ | v2 能量明细：月令、通根、合冲、党势（可视化 + reasons 逐条） |
| 叁·性情运作 | 这让我在生活里是什么样的人？ | 十神画像、双副轴、社交模式（现有内容重组，全部回扣主线） |
| 肆·关系姿态 | 我在关系里的出厂设置？ | 依恋、主动性、表达（从现有六维中拆出关系相关项） |
