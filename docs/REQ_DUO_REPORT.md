## 三、双人标签库 v1（草案——规则表映射，确定性，你逐条审）

> 双人标签由两张盘的指标**组合**触发（差值、配对、互补），与个人标签不同源。
> 出牌机制同个人版：按幅度排序每章取 2~3 个。

**缘起域（吸引结构）**：

| 标签 | 触发（示意） |
|---|---|
| 双向奔赴 | 双方主动性均 ≥55 |
| 一追一逃 | 主动性差 ≥25 且 一方回避型 |
| 被吸住的观察员 | 一方警觉高慢热 × 另一方直球高表达 |
| 互为补给 | 五行喜忌互补 tone=mutual（TA 旺的是你喜的，反之亦然） |
| 单向充电宝 | 互补 tone=oneway |
| 同类相认 | 主轴十神组相同 |
| 镜像互补 | 主轴十神组互为生克对位（如印遇食伤） |

**相处域（日常样态）**：

| 标签 | 触发 |
|---|---|
| 同频对讲机 | 表达意愿均 ≥55 且差 ≤15 |
| 一个广播一个收音 | 表达差 ≥25 |
| 静音电台 | 表达均 ≤45（默契型，也可能闷出事） |
| 时差恋人 | 关系节奏一快一慢（trust_speed 差 ≥25） |
| 黏度不对等 | 依赖差 ≥25 |
| 各自有房间 | 双方自主 ≥58（独立型组合） |
| 有人掌舵有人划桨 | 主动性差 15~25 且控制差 ≥15（分工明确，非失衡） |

**摩擦域（冲突结构）**：

| 标签 | 触发 |
|---|---|
| 冷战二人组 | 双方冲突表达 ≤45 |
| 点火就着秒和好 | 双方冲突表达 ≥58 且 稳定均 ≥55 |
| 一个爆一个熄 | 冲突表达差 ≥25 |
| 谁都不低头 | 双方控制 ≥60 |

| 确认与空间之争 | 一方焦虑型 × 一方回避型（经典错位） |

**长线域（经营与时运）**：

| 标签 | 触发 |
|---|---|
| 双补给期 | 两人当前大运均为补给/偏强 |

## 四、对比数据表征（图表规格）

1. **双色对比条**（每章核心图）：同一指标两人并排——上条 A（青）、下条 B（绯），
   右侧差值签（「差 27 · 显著」/「差 6 · 同步」）。各章指标组：
   - 缘起：浪漫倾向 / 新鲜感需求 / 信任建立速度
   - 相处：表达意愿 / 情感依赖 / 沟通需求
   - 摩擦：冲突表达 / 关系警觉 / 边界控制
   - 长线：情绪稳定 / 自主空间 + **双人大运补耗并排条**（两行，当前段高亮）
2. **五行互补签**（缘起章）：elementSynergy 两侧文案已有，配「A 最旺→B 喜/忌」箭头小图。
3. **六维雷达引用**（封面或壹章）：沿用关系剧本现有六维数据，缩略展示，总分是否上封面待拍板。

## 五、AI 叙述层

- **输入** = buildRelationshipFacts（已含 spine 主线、契约、六维、行为断语）
  + 新增 duoTags（四域标签及判定指标）+ 对比数据。
- **输出 JSON**：`{"headline":"关系一句话判词，15字内","pages":{"origin":{"essay","advice"},"daily":{...},"friction":{...},"longrun":{...}}}`
- **铁律**（同个人版 + 双人特有）：
  1. 事实只来自清单；每章围绕该域标签展开，且须回扣 spine 主线；
  2. 正文禁数字禁命理术语（长线章可提年龄段）；
  3. 以名字称呼两人，写「你们」的具体场景（谁先发消息、冷战谁破冰），
     禁止各写一段变成两份个人报告的拼贴；
  4. 建议必须落在「两个人一起能做的一件事」上。
- **校验器**：结构完整、字数达标、数字与黑话检查（同个人版机制）；
  **兜底**：确定性四章模板（duoTags + TAG_EXPLAIN + 现有 behaviors/hotspots 素材改写）。
