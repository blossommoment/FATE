## 五、AI 叙述层

- **输入** = buildRelationshipFacts（已含 spine 主线、契约、六维、行为断语）
  + 新增 duoTags（五域标签及判定指标）+ 对比数据。
- **输出 JSON**：`{"headline":"关系一句话判词，15字内","pages":{"origin":{"essay","advice"},"daily":{...},"friction":{...},"longrun":{...},"season":{...}}}`（长线/时运两章分开写：长线=怎么经营，时运=什么时候）
- **铁律**（同个人版 + 双人特有）：
  1. 事实只来自清单；每章围绕该域标签展开，且须回扣 spine 主线；
  2. 正文禁数字禁命理术语（时运章可提年龄段与年份）；
  3. 以名字称呼两人，写「你们」的具体场景（谁先发消息、冷战谁破冰），
     禁止各写一段变成两份个人报告的拼贴；
  4. 建议必须落在「两个人一起能做的一件事」上。
- **校验器**：结构完整、字数达标、数字与黑话检查（同个人版机制）；
  **兜底**：确定性五章模板（duoTags + TAG_EXPLAIN + 现有 behaviors/hotspots 素材改写）。
